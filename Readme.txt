關於玩家控制：
1.沒有fast狀況下：

基本出牌：
	輪到你出牌時，先按"Next"產生按鈕，點選按鈕選擇要出的排，再按"Next"即可出牌。
選色出牌：
	輪到你出牌時，先按"Next"產生按鈕，點選按鈕選擇要出的排，再按"Next"產生顏色按鈕，點選顏色按鈕選擇顏色後，再按"Next"即可出牌。
抽排：
	選擇要出的牌時，點選最左邊藍色按鈕，為抽牌選項。

2.有fast狀況下：

一率自動產生按鈕，按下後立即出牌/選色。
(方便但你會錯過其他玩家的流程)

****注意****
一旦出錯牌，遊戲就崩潰了，按"reset"重置。
